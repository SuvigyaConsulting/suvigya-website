# SUVIGYA CONSULTING - Light Theme Version

A professional light-themed version of the SUVIGYA CONSULTING website with beige and earth tones.

## Features

This is the **light theme** version with:
- Beige and earth tone color palette
- Professional, clean aesthetic
- Light background with warm accents
- All the same advanced features as the dark version

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Quick Start

**Windows**: Double-click `START.bat`

**Mac/Linux**: Run `node serve.js` (after building with `npm run build`)

## Color Scheme

- **Background**: Beige tones (#fefdfb to #806746)
- **Primary**: Earth tones (#a8906f to #312617)
- **Accents**: Warm browns and tans
- **Text**: Dark slate on light background

## Differences from Dark Version

- Light beige background instead of dark slate
- Earth tone gradients instead of blue/cyan
- Professional, corporate aesthetic
- Better for print and formal presentations
